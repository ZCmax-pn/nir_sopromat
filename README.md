# Sopromat React App

Расчёт статически определимых рам и балок - React приложение с адаптивным дизайном.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)

## Сборка для продакшена

```bash
npm run build
```

## Особенности

- ✅ Полностью переписано на React
- ✅ Адаптивный дизайн для всех устройств
- ✅ Поддержка touch-событий для мобильных устройств
- ✅ Модульная архитектура компонентов
- ✅ Использование React Hooks

## Структура проекта

```
src/
  ├── components/     # React компоненты
  ├── hooks/          # Custom React hooks
  ├── App.js          # Главный компонент
  └── index.js        # Точка входа
```

## Технологии

- React 18
- CSS3 (адаптивный дизайн)
- Canvas API для отрисовки
