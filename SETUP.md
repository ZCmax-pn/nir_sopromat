# Инструкция по установке и запуску

## Установка зависимостей

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

Приложение откроется автоматически в браузере по адресу http://localhost:3000

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `build/`

## Структура проекта

```
├── public/              # Статические файлы (HTML, изображения)
│   ├── images/          # Изображения опор
│   └── index.html        # HTML шаблон
├── src/
│   ├── components/       # React компоненты
│   │   ├── Header.js
│   │   ├── Workspace.js
│   │   ├── ConstructionCanvas.js
│   │   ├── FormulaSection.js
│   │   ├── ResultsSection.js
│   │   └── ...
│   ├── hooks/           # Custom React hooks
│   │   └── useConstruction.js
│   ├── App.js           # Главный компонент
│   └── index.js         # Точка входа
└── package.json         # Зависимости проекта
```

## Особенности

- ✅ Полностью переписано на React с использованием Hooks
- ✅ Адаптивный дизайн для всех устройств (desktop, tablet, mobile)
- ✅ Поддержка touch-событий для мобильных устройств
- ✅ Модульная архитектура компонентов
- ✅ Оптимизированная отрисовка через Canvas API

## Адаптивность

Приложение автоматически адаптируется под:
- Десктоп (1200px+)
- Планшеты (768px - 1200px)
- Мобильные устройства (до 768px)
- Маленькие экраны (до 480px)
